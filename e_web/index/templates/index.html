Вопросы и Ответы

Что такое Easy Code?

Это полностью бесплатная платформа для изучения основ программирования с нуля. Easy Code был создан разработчиками и сообществом Хекслет для того, чтобы любой желающий мог попробовать себя в программировании и заложить правильный фундамент для новой профессии, основанный не на запоминании специфики, а на понимании системы в целом.

А если что-то не получается?

Специалисты и учащиеся у нас есть раздел «Обсуждения». Здесь можно задать интересующий вас вопрос или помочь ответить на вопрос другого пользователя. Обсуждения доступны для всех зарегистрированных пользователей.

Чему здесь можно научиться?

Вы можете познакомиться с основами языков JavaScript, PHP, Java, Python, Ruby, Go, C# и другими. Также на Code Basics доступны курсы по HTML и CSS.

Как начать обучение?

Достаточно просто зарегистрироваться. Регистрация на платформе бесплатная, после регистрации вы получите доступ ко всем курсам Code Basics. Выбирайте любой – и проходите.
