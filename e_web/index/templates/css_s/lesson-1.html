Информация про CSS

Преимущества CSS очевидны, так как он позволяет оформлять сайты: из некрасивой HTML

CSS позволяет добавлять не только стили для блоков, но также позволяет создавать
анимацию, что может проигрываться при загрузке страницы или же при наведении
мыши на объект.
