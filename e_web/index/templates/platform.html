О проекте


Что такое Easy Code?

Это открытый бесплатный проект для изучения основ программирования.
Интерактивные курсы с практикой рассчитаны на тех, кто только начал интересоваться разработкой.
На Easy Code уже есть курсы по PHP, Java, JS, Python, Ruby, HTML, CSS, Racket и Go.

В 2020 году проект Easy Code занял 2 место на Международном конкурсе
открытых онлайн-курсов EdCrunch Award OOC в номинации «Лучший внеплатформенный онлайн-курс».

Почему мы его создали?

Большинство доступных материалов для новичков больше похожи на справочники или туториалы в стиле «повтори за учителем».

Цель проекта Easy Code — грамотный старт.
Мы хотим научить начинающих программистов лучшим стандартам кодирования,
правильному именованию функций, пониманию систем, а не запоминанию специфики.

Для кого наш проект?

1. Для новичков, которые только начинают знакомиться с синтаксисом языков программирования и базовыми алгоритмами.
2. Для начинающих программистов, которые «плавают» в основах и хотят узнать больше, чем написано в учебниках.
3. Для учителей, работающих с новичками.
